
=======================
Robotice Mobile Client
=======================

Hybrid mobile client for https://github.com/robotice/robotice-control and https://github.com/robotice/robotice


Usage
-----
Development in browser

.. code-block:: bash

    ionic serve

Build app for platform

.. code-block:: bash

    localdevel -d
